# Manual de Operación CNC para PCBs  
## Roland SRM-20

Este repositorio contiene el Manual de Operación para el Mecanizado CNC de Placas de Circuito Impreso (PCB) utilizando la Roland SRM-20, desarrollado para su uso en entornos académicos y FabLabs.

## Contenido

- Flujo completo de trabajo CAD/CAM/CNC
- Generación de archivos Gerber y Excellon
- Uso de FlatCAM y VPanel
- Parámetros reales de mecanizado para PCB
- Seguridad, EPP y control de riesgos (FR-1/FR-2, FR-4, CEM, etc.)
- Checklist de errores comunes

## Equipamiento

- Fresadora CNC Roland SRM-20
- FlatCAM (CAM)
- KiCad 9 / Proteus 8 (CAD)
- PC con Windows

## Seguridad

Este manual enfatiza el cumplimiento de:
- DS N°594
- Ley 16.744
- Uso obligatorio de EPP para mecanizado de placas PCB
